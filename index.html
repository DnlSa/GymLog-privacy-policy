Informativa sulla Privacy di MacroDiary

Ultimo aggiornamento: Giugno 2026

La presente Informativa sulla Privacy descrive le modalità con cui MacroDiary gestisce e tutela i dati degli utenti. L'applicazione è progettata mettendo al primo posto la riservatezza e il controllo totale dei dati da parte dell'utente.

1. Tipologia di dati e Archiviazione Locale

L'applicazione funziona come un client locale ("guscio"). Non raccoglie, non memorizza e non condivide alcun dato personale, biometrico, sanitario o diario alimentare dell'utente su server di nostra proprietà. Tutti i dati inseriti (inclusi i pasti, i macronutrienti e gli obiettivi) vengono salvati esclusivamente nella memoria locale del dispositivo dell'utente in modo sicuro.

2. Gestione delle API Key e Servizi di Terze Parti

L'utente ha la possibilità di configurare le proprie chiavi API personali per usufruire delle funzionalità di Intelligenza Artificiale.

  • Le chiavi API e gli indirizzi dei server inseriti vengono salvati esclusivamente in locale sul dispositivo.
  • Le richieste IA vengono inviate dall'app direttamente agli endpoint configurati dall'utente (es. OpenAI, Anthropic, o server Ollama locale), senza alcuna intermediazione da parte nostra.
  • Interagendo con tali servizi, i dati relativi ai prompt inviati saranno soggetti alle rispettive policy sulla privacy, che ti invitiamo a consultare:
    • Informativa sulla Privacy di OpenAI
    • Informativa sulla Privacy di Anthropic

3. Diritti dell'utente e Cancellazione dei Dati

Poiché non memorizziamo alcun dato su server esterni, non abbiamo la possibilità di visualizzare, modificare o cancellare i tuoi dati. L'utente ha il pieno ed esclusivo controllo delle proprie informazioni. Per eliminare definitivamente tutti i dati dall'applicazione e le API Key salvate, è sufficiente:

  • Utilizzare la funzione di reset dentro l'app (se presente).
  • Cancellare i dati dell'applicazione dalle impostazioni di Android o disinstallare l'applicazione dal dispositivo.

4. Modifiche alla presente Informativa

Ci riserviamo il diritto di modificare questa informativa in qualsiasi momento per adeguarla a future funzionalità dell'applicazione. Eventuali modifiche saranno pubblicate su questa pagina.
