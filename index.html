Informativa sulla Privacy

Questa applicazione è un client locale ("guscio") e non raccoglie, memorizza o condivide alcun dato personale o sanitario dell'utente su server di nostra proprietà.

Gestione delle API Key e Dati

Tutte le chiavi API e gli indirizzi dei server inseriti dall'utente vengono salvati esclusivamente in locale sul dispositivo in modo sicuro. Le richieste IA vengono inviate direttamente dall'app agli endpoint configurati dall'utente (es. OpenAI, Anthropic, server Ollama locale).
